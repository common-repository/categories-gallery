Show all posts product categories in gallery format.

== Description ==
This simple plugin lets you show all post categories in gallery format.
Use shortcode: "[categoriesgallery]" in your post or page.
If any issues regarding this plugin mail me @: sandeep.tete2013@gmail.com


== Installation ==
- Upload zip file.
- Activate the plugin.
- Scroll down to 'categories gallery'.
- Modify the settings.
- Done.
